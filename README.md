# kangeki-schedule
観劇スケジュール
纯个人偏好的，观剧规划日程表，主要收录宝塚歌劇団各组公演和部分感兴趣的外部音乐剧日程。如需要追加其他剧目日程，可以自行下载网页修改代码。推荐配合Claude+google calendar使用。
使用方法&注意事项：
1.本网页依赖本地缓存，只要不清空本地缓存，收藏列表就不会清空。建议将此网页作为一个规划行程的中转站，规划好后交给AI写入自己的电子日程表。
2.上方标签可筛选组别、公演地域，点击具体日期可以看到当日所有演出。在演出条目可以点击♡收藏公演。已收藏公演会在日历上高亮表示。
3.每场公演点选收藏后可添加memo，请自行填入各种需要记录的备忘信息（如取票方式、貸切公演时间等）。
4.お気に入りリスト可以点击复制指令交给已经claude，让小克帮你写入系统自带schedular或已连接的google calendar。
